## Long Description
*(max 4000 characters — no HTML, no keyword stuffing)*

Not2Pix is a pixel art editor built for mobile. Whether you are drawing sprites, tilesets, or frame-by-frame animations, it gives you precise control on a touchscreen without getting in the way.

**Drawing tools**

- Pencil, fill, eyedropper, eraser, and line tools
- Mirror drawing on X and Y axes simultaneously
- Selection tool with move, rotate, and import-from-image support
- Background trace tool to draw over a reference image

**Layers**

Work with multiple layers per frame. Toggle visibility, reorder, merge, or delete layers at any time. Each layer is independent so you can keep linework, color fills, and details separate.

**Animation**

Add frames and play them back at a custom frame rate. Onion skinning shows the previous frame as a ghost so you can keep motion consistent. Export the finished animation as a GIF with one tap.

**Color**

An HSV color picker with a hex input field lets you dial in exact colors. Build and save a custom palette, or import an existing one from a GIMP palette (.gpl) or plain hex list. Export your palette to share it or reuse it in other tools.

**File support**

- Open and save PNG files
- Open and save Aseprite (.ase) files for roundtrip compatibility with desktop tools
- Export flattened PNG or animated GIF
- Import any image as a selection to paste and position on the canvas

**Minimap and navigation**

A live minimap gives you a full-canvas overview while you zoom in to work. Pan and zoom with standard pinch and drag gestures.

**Customizable layout**

Move the toolbar and color bar to any vertical position on screen so they stay out of your way on any device size.

**Designed for Android**

Not2Pix integrates with Android's file picker and Storage Access Framework, so it works with files from local storage, cloud drives, and other apps. It also accepts edit requests from compatible tiling apps for a seamless external-editor workflow.

No ads. No account required.
